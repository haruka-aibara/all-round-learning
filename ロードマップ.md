# DevOpsロードマップ・チェックリスト
こちらは AWS を主に使う場合の DevOps エンジニアとしての学習ロードマップを
Claude 3.7 Sonnet 君に出力してもらったものです。

自身が何を知っており、何を知らないか、知らないところを学ぶ前提として何があるか（例えば、Kubernetesを知る前にDockerの知識が前提）など一目で分かるリストが欲しくて作成

「知っている」の基準は、その内容を自分の言葉で説明できる状態になっているか　とします。
