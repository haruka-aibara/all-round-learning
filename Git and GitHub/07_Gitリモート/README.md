# Gitリモート

リモートリポジトリは、Gitの重要な機能の1つです。ローカル（自分のコンピュータ）のリポジトリと、GitHubなどのサーバー上にあるリポジトリを連携させることで、チーム開発やコードの共有を可能にします。例えば、複数の開発者が同じプロジェクトで作業する際に、リモートリポジトリを通じて変更を共有することができます。

このセクションでは、以下の基本的なリモート操作を学びます：

1. **リポジトリのクローン**
   - リモートリポジトリの取得
   - クローンの方法（HTTPS/SSH）
   - クローン後の設定

2. **リモートの管理**
   - リモートの追加と削除
   - リモートの一覧表示
   - リモートの設定変更

3. **変更の同期**
   - プッシュ（ローカル→リモート）
   - プル（リモート→ローカル）
   - フェッチ（変更の取得）

4. **認証とセキュリティ**
   - パーソナルアクセストークン（PAT）
   - SSH鍵の設定
   - セキュアな接続方法
