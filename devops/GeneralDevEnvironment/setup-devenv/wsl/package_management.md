## パッケージの更新とアップグレード


1. 以下のコマンドを WSL に入力する。
```
sudo apt update && sudo apt upgrade
```

![image](https://github.com/user-attachments/assets/7205def2-0717-4404-b7d7-0c6d9f02971e)

2. パスワードを入力する。
![image](https://github.com/user-attachments/assets/36989652-36ca-4774-af2f-13cfd098bdf8)

3. 次の文言が表示されます。
```
Do you want to continue? [Y/n]
```
![image](https://github.com/user-attachments/assets/18daa546-bfc2-435d-9684-9dd991c02058)

4. 「Y」を入力し Enter キーを押下する。
```
Do you want to continue? [Y/n] Y
```
![image](https://github.com/user-attachments/assets/513eaf8a-eda7-4c9e-9d10-795d1ad71db4)

5. 処理が流れ終わったら完了です。
![image](https://github.com/user-attachments/assets/d1193b17-c394-49e7-98be-4acc51f83152)
